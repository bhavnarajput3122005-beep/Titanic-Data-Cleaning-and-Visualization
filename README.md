# Titanic Data Cleaning & Visualization Project

## Overview
This project focuses on cleaning, processing, and visualizing the Titanic dataset using Python. The goal was to transform raw data into meaningful insights through data preprocessing and visualization techniques.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Data Cleaning Steps
- Handled missing values in Age and Embarked columns
- Removed duplicate records
- Dropped the Cabin column due to excessive missing values

## Visualizations Created
1. Survival Count
2. Survival by Gender
3. Age Distribution
4. Passenger Class Distribution

## Key Findings
- More passengers did not survive than survived.
- Female passengers had a significantly higher survival rate.
- Most passengers were between 20 and 40 years old.
- Third-class passengers formed the largest group of travelers.

## Outcome
This project helped develop skills in data cleaning, exploratory data analysis (EDA), data visualization, and storytelling with data.
